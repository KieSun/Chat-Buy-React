## Run Project
``` bash
# clone 
git clone https://github.com/KieSun/Chat-Buy-React.git
cd chat-buy-react

# Mac install MongoDb
brew install mongodb

# run MongoDb
mongod --config /usr/local/etc/mongod.conf

# connect mongo
mongo

# install npm package
npm install

# run server
npm run server

# run localhost
npm run start
```
